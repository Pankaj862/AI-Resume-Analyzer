# 🚀  Resume Analyzer & Interview Preparation Platform

An AI-powered platform that analyzes resumes against job descriptions, identifies skill gaps, generates personalized interview questions, and creates a structured learning roadmap to help users prepare for technical interviews.

## 🌐 Live Demo

Frontend: https://resume-analyzer-ten-zeta.vercel.app

---

## ✨ Features

### 📄 Resume Analysis
- Upload Resume (PDF/DOCX)
- Paste Job Description
- AI-powered resume evaluation
- Match score calculation

### 🎯 Skill Gap Detection
- Identifies missing skills
- Highlights strengths and weaknesses
- Compares resume against job requirements

### 💡 Technical Interview Questions
- Generates personalized interview questions
- Provides expected answers
- Focuses on role-specific technologies

### 📈 Learning Roadmap
- AI-generated preparation roadmap
- Daily learning plan
- Skill improvement recommendations

### 🔐 Authentication
- User Registration
- Login / Logout
- JWT Authentication
- Protected Routes
- Secure HTTP-only Cookies

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Pankaj862/AI-Resume-Analyzer.git

cd AI-Resume-Analyzer
```

### Backend Setup

```bash
cd Backend

npm install

npm start
```

### Frontend Setup

```bash
cd Frontend

npm install

npm run dev
```

## 🔄 Workflow

1. Register/Login
2. Upload Resume
3. Paste Job Description
4. Generate Interview Strategy
5. Review:
   - Match Score
   - Skill Gaps
   - Technical Questions
   - Learning Roadmap
